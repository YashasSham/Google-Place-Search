Place Finder App

This web application lets you find the places near your location. You have the option to type in the places you want to search or speak it.

Permission: You are prompted to allow location and microphone. Please give permission by clicking 'Allow'
    Say-
  1. "Search XXXXX": to put any word(XXXXX) in the searchbox and press enter to search.
  2. "Number (1-20)": to select the list elements based on the number next to each listing
  3. "Next": to go to the next list element
  4. "Please Reload": to reload the page, askes for the permissions
  5. "Please Close": to close the window permanently
